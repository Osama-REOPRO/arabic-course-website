# Allgemeine Nomen vs. Eigennamen

## Eine interessante Beobachtung

Wenn wir uns die Beispiele von oben ansehen, merken wir etwas Interessantes. Manche Nomen brauchen einen **bestimmten Artikel**, damit sie bestimmt sind:
- **das Buch** (_the book_)
- **der Lehrer** (_the teacher_)
- **die Schule** (_the school_)

Ohne den Artikel meinen diese Wörter normalerweise keine bestimmte Sache oder Person. Aber andere Nomen sind **von alleine bestimmt**. Sie brauchen keinen Artikel:
- **Anna**
- **Berlin**
- **Deutschland**
- **BMW**

Diese Wörter zeigen sofort eine ganz genaue Person, einen Ort oder eine Firma, weil es ihr Name ist. Hier ist eine wichtige Frage:

> Warum brauchen manche Nomen einen Artikel, um bestimmt zu sein, und andere nicht?

Die Antwort ist der Unterschied zwischen **allgemeinen Nomen** (Gattungsnamen) und **Eigennamen**. Allgemeine Nomen meinen eine ganze Gruppe von Dingen. Sie brauchen meistens einen Artikel, wenn sie bestimmt sein sollen. Eigennamen meinen eine ganz genaue, einzigartige Sache. Sie sind **schon von Natur aus bestimmt**.

## Allgemeine Nomen vs. Eigennamen

Wir können Nomen in zwei große Gruppen teilen: **allgemeine Nomen** und **Eigennamen**.

### Allgemeine Nomen

Ein **allgemeines Nomen** (Gattungsname) meint eine **allgemeine Gruppe oder Klasse von Dingen**, nicht ein einzelnes Ding. Allgemeine Nomen können **unbestimmt** (irgendein Ding aus der Gruppe) oder **bestimmt** (ein ganz genaues Ding aus der Gruppe) sein. Das kommt auf den Artikel und die Situation an.

Beispiele auf Deutsch:
- **Buch** (book)
- **Hund** (dog)
- **Schule** (school)
- **Auto** (car)
- **Lehrer** (teacher)

Diese Nomen kann man mit verschiedenen Artikeln benutzen:

- **ein Buch** – *a book* (irgendein Buch)
- **das Buch** – *the book* (ein ganz bestimmtes Buch)

- **ein Hund** – *a dog*
- **der Hund** – *the dog*

Das Wort allein sagt noch nicht, welches Ding gemeint ist. Das zeigt uns erst der Artikel.

### Eigennamen

Ein **Eigenname** ist der Name von einer **ganz bestimmten, einzigartigen Person, einem Ort, einer Firma oder einer Sache**. Er beschreibt keine Gruppe, sondern nennt ein genaues Ding beim Namen.

Beispiele auf Deutsch:
- **Anna**
- **Deutschland**
- **Berlin**
- **BMW**
- **Europa**

Eigennamen sind fast immer **von alleine bestimmt**. Wenn man **Berlin** sagt, meint man genau diese eine Stadt und nicht irgendeine Stadt auf der Welt.

## Zusammenfassung

| Allgemeine Nomen | Eigennamen |
|--------------|--------------|
| Meinen eine allgemeine Gruppe | Meinen eine einzigartige Sache oder Person |
| Können bestimmt oder unbestimmt sein | Sind fast immer von alleine bestimmt |
| Beispiele: **Buch, Hund, Schule, Auto** | Beispiele: **Berlin, Deutschland, Anna, BMW** |

Kurz gesagt: **Allgemeine Nomen beschreiben, was für eine Art von Sache etwas ist. Eigennamen sagen uns den genauen Namen von einer Sache.**
