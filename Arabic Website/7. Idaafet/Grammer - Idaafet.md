# Idaafet

**Grundidee:** Man kann zwei unbestimmte Nomen kombinieren, um etwas genauer zu beschreiben (ähnlich wie ein Adjektiv). Das zweite Nomen beschreibt das erste näher. Das nennt man **"idaafet"**. Das ist dem deutschen Kompositum sehr ähnlich, nur anders aufgebaut.

**Aussprache-Änderung:** Endet das erste Wort auf -et/-at (Taa2 Marbuutat), wird das t ausgesprochen (-ett/-att).
