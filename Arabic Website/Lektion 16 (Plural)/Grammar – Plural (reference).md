# The Plural (Jam3) – Reference

Arabic has **three** kinds of plural.

| Plural type | Ending | When? | Example |
| --- | --- | --- | --- |
| **Sound masculine** | **-iin** | male person (profession, origin, state); also mixed groups | mwazzaf → mwazzaf**iin** |
| **Sound feminine** | **-aat** | words ending in **-et/-at** (Taa Marbouta); often new/imported words too | sayyaarat → sayyaar**aat** |
| **Broken** | *no fixed pattern* | most basic words, objects, animals, many person-words → **memorise** | walad → **wlaad** |

## Decision guide

1. Does the word end in **-et/-at**? → very likely **-aat**
2. Male profession/origin/state, ending in a consonant? → likely **-iin**
3. Anything else → likely **broken** → learn it as you go

> [!NOTE] Important
> The broken plural has **no** simple rule – native speakers just know it (like *child → children* in English). For new/foreign words, **-aat** is a good default guess.

## Common broken plurals

| Singular | Plural | English |
| --- | --- | --- |
| beet | byuut | houses |
| ktaab | kutub | books |
| walad | wlaad | boys |
| bint | banaat | girls |
| tifl | atfaal | children |
| kalb | klaab | dogs |
| meftaa7 | mafaatii7 | keys |

Possessive suffixes attach to plurals too: mumarridiin + i = **mumarridiini** (*my nurses*).
