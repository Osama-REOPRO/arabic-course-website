# Wurzel & Muster (Jazr & Wazn) – Referenz

Arabische Wörter bestehen aus zwei Teilen:

> **Wurzel** = die *Bedeutung* (meist 3 Konsonanten) · **Muster (Wazn)** = die *Funktion / Wortart*

Man „gießt" eine Wurzel in ein Muster und erhält ein fertiges Wort.

## Der Platzhalter f-3-l

Grammatiker markieren die drei Wurzelbuchstaben mit **f · 3 · l** (Wurzelbuchstabe 1 · 2 · 3).
Statt `[W1]a[W2]a[W3]` schreibt man einfach **fa3al**.

## Die wichtigsten Muster

| Muster | Funktion | Beispiel (Wurzel → Wort) |
| --- | --- | --- |
| **fa3al** | „er hat X getan" (Vergangenheit) | k-t-b → **katab** (er schrieb) |
| **maf3al** / **maf3alet** | ein **Ort**, an dem X passiert | k-t-b → **maktab** (Büro); d-r-s → **madraset** (Schule) |
| **fa33aal** / **fa33aalet** | **Person/Maschine**, die X (beruflich) tut | T-b-5 → **Tabbaa5** (Koch); gh-s-l → **ghassaalet** (Waschmaschine) |
| **faa3el** | derjenige, **der** X tut | k-t-b → **kaateb** (Schriftsteller) |
| **maf3uul** | etwas, dem X **angetan** wurde | k-t-b → **maktoob** (Brief); f-t-7 → **maftuu7** (offen) |
| **mif3aal** | **Werkzeug**, mit dem man X tut | f-t-7 → **miftaa7** (Schlüssel) |

## Beispiel: eine Wurzel, viele Muster

**T-b-5** (Idee von „Kochen"): Taba5 (er kochte) · maTba5 (Küche) · Tabbaa5 (Koch)

> [!NOTE] Wichtig
> **Nicht jedes Muster passt zu jeder Wurzel.** Man kann nicht beliebig kombinieren – manche Kombinationen ergeben kein echtes Wort. Mit der Zeit bekommt man ein Gespür dafür.
