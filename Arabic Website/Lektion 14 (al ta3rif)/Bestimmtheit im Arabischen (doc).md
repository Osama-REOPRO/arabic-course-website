Die Konzepte von **Gattungsnamen**, **Eigennamen** und **Bestimmtheit** existieren auch im Arabischen.

Genau wie im Deutschen und Englischen beziehen sich **Eigennamen** auf einzigartige Entitäten und sind daher in der Regel **von Natur aus bestimmt**. Sie benötigen normalerweise keinen Artikel.

Beispiele:

- **أحمد** (_Ahmad_)
- **دمشق** (_Damascus_ / Damaskus)
- **سوريا** (_Syria_ / Syrien)

Diese Namen sind bereits bestimmt, da sie eine spezifische Person oder einen bestimmten Ort identifizieren.

**Gattungsnamen** hingegen beziehen sich auf allgemeine Kategorien. Sie können entweder **bestimmt** oder **unbestimmt** sein, und das Arabische unterscheidet zwischen diesen beiden Formen mithilfe des **bestimmten Artikels** **الـ** (_al-_).

Beispiele:

- **كتاب** (_a book_ / ein Buch)
    
- **الكتاب** (_the book_ / das Buch)
    
- **بيت** (_a house_ / ein Haus)
    
- **البيت** (_the house_ / das Haus)
    

Im Gegensatz zum Englischen und Deutschen gibt es im Arabischen **nur einen bestimmten Artikel**. Es existiert **kein unbestimmter Artikel**, der dem englischen _a/an_ oder dem deutschen _ein/eine_ entspricht.

Stattdessen wird ein Gattungsname **standardmäßig als unbestimmt** verstanden, wenn er **nicht** das Präfix **الـ** (_al-_) trägt. Das Hinzufügen von **الـ** macht das Nomen bestimmt.

Dies führt zu einer sehr einfachen Regel:

- **Gattungsname + الـ → Bestimmt**
    
- **Gattungsname ohne الـ → Unbestimmt**
    
- **Eigenname → Normalerweise bestimmt ohne jeglichen Artikel**
    

Zum Beispiel:

| Arabisch | Bedeutung |
|---|---|
| **كتاب** | _a book_ / ein Buch |
| **الكتاب** | _the book_ / das Buch |
| **أحمد** | _Ahmad_ (bereits bestimmt) |
| **دمشق** | _Damascus_ / Damaskus (bereits bestimmt) |

Mit anderen Worten: Das Arabische drückt Bestimmtheit primär dadurch aus, dass **الـ** an Gattungsnamen angefügt wird, während das Fehlen von **الـ** im Allgemeinen anzeigt, dass ein Gattungsname unbestimmt ist.
