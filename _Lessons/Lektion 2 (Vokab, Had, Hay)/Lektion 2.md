## Beispiele
1) بيت (house)
Arabic: هاد بيت
German: Das ist ein Haus.
English: This is a house.
2) صبي (boy)
Arabic: هاد صبي
German: Das ist ein Junge.
English: This is a boy.
With emphasis:
German: Dieser Junge.
English: This boy.

3) رجال (men)
Arabic: هدول رجال
German: Das sind Männer.
English: These are men.
4) كلب (dog, masculine)
Arabic: هاد كلب
German: Das ist ein Hund.
English: This is a dog.
With emphasis:
German: Dieser Hund.
English: This dog.

5) علبة (box, feminine)
Arabic: هاي علبة
German: Das ist eine Dose.
English: This is a box.
With emphasis:
German: Diese Dose.
English: This box.

6) صبية (girl)
Arabic: هاي صبية
German: Das ist ein Mädchen.
English: This is a girl.
With emphasis:
German: Dieses Mädchen.
English: This girl.

7) كلبة (female dog)
Arabic: هاي كلبة
German: Das ist eine Hündin.
English: This is a female dog.
With emphasis:
German: Diese Hündin.
English: This female dog.

8) مرا (woman)
Arabic: هاي مرا
German: Das ist eine Frau.
English: This is a woman.
With emphasis:
German: Diese Frau.
English: This woman.

9) حب (love)
Arabic: هاد حب
German: Das ist Liebe.
English: This is love.
