in Syria, the 'e' in 'el-' is very very short, you can barely hear it
unless it is a sun letter 'e', in which case it is much stronger
# feminine words before "al-" within an idafeh
within an idafeh, if we add "al" to the second word, and the first word is feminine "ending with -et", then the "e" is removed and only the "-t" remains, this makes pronunciation much easier
Talbet + Madraset = Talebt el-madraset
# pronounce an idafeh like a single word
don't put any space in between, an idafeh with an 'al-' in between flows and sticks together like a single longer word
mudif + el + maT3am = mudif-el-maT3am
