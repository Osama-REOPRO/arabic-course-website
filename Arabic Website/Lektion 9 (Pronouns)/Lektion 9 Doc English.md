# Possessive pronouns
in Arabic there are no separate "posessive pronouns" like (my, your, their, etc.), instead, suffixes are added to the end of the word, those suffixes are listed below

| Pronomen              | Suffix   | Beispiel | Deutsch    |
| :-------------------- | :------- | :------- | :--------- |
| **ana** (ich)         | **-i**   | ismi     | mein Name  |
| **inte** (du, m.)     | **-ak**  | ismak    | dein Name  |
| **inti** (du, w.)     | **-ek**  | ismek    | dein Name  |
| **hwwe** (er)         | **-o**   | ismo     | sein Name  |
| **hyye** (sie)        | **-a**   | isma     | ihr Name   |
| **ni7na** (wir)       | **-na**  | ismna    | unser Name |
| **intu** (ihr)        | **-kon** | ismkun   | euer Name  |
| **hinnin** (sie, pl.) | **-on**  | ismun    | ihr Name   |
