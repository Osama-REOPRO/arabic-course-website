# Der Plural (Jam3) – Referenz

Arabisch hat **drei** Pluralarten.

| Pluralart | Endung | Wann? | Beispiel |
| --- | --- | --- | --- |
| **Gesund maskulin** | **-iin** | männliche Person (Beruf, Herkunft, Zustand); auch gemischte Gruppen | mwazzaf → mwazzaf**iin** |
| **Gesund feminin** | **-aat** | Wörter auf **-et/-at** (Taa Marbouta); oft auch neue/importierte Wörter | sayyaarat → sayyaar**aat** |
| **Gebrochen** | *kein festes Muster* | die meisten Grundwörter, Objekte, Tiere, viele Personenwörter → **auswendig lernen** | walad → **wlaad** |

## Entscheidungshilfe

1. Endet das Wort auf **-et/-at**? → sehr wahrscheinlich **-aat**
2. Männlicher Beruf/Herkunft/Zustand, endet auf Konsonant? → wahrscheinlich **-iin**
3. Alles andere → wahrscheinlich **gebrochen** → mitlernen

> [!NOTE] Wichtig
> Für den gebrochenen Plural gibt es **keine** einfache Regel – Muttersprachler kennen ihn auswendig (wie *Kind → Kinder* im Deutschen). Bei neuen/fremden Wörtern ist **-aat** ein guter Standard-Tipp.

## Häufige gebrochene Plurale

| Singular | Plural | Deutsch |
| --- | --- | --- |
| beet | byuut | Häuser |
| ktaab | kutub | Bücher |
| walad | wlaad | Jungen |
| bint | banaat | Mädchen |
| tifl | atfaal | Kinder |
| kalb | klaab | Hunde |
| meftaa7 | mafaatii7 | Schlüssel |

Possessivsuffixe hängt man auch an Plurale an: mumarridiin + i = **mumarridiini** (*meine Krankenpfleger*).
