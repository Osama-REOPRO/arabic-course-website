# al- / Definiteness – Reference

> [!NOTE] Core rule
> Arabic has **only one** article: the definite article **el- / al-** (ال).
> There is **no** indefinite article.

| Form | Meaning |
| --- | --- |
| noun **+ el-** | definite (*the*) |
| noun **without el-** | indefinite (*a / an*) |
| **proper noun** (Ahmad, Damascus) | already definite → no el- |

- **ktaab** → *a book* · **el-ktaab** → *the book*
- **beet** → *a house* · **el-beet** → *the house*

## Sun vs. moon letters

The **l** of el- behaves differently depending on the first letter of the noun.

| | Rule | Example |
| --- | --- | --- |
| **Sun letter** | the "l" drops, first letter is **doubled** | el + shams → **esh-shams** |
| **Moon letter** | el- stays normal | el + 2amar → **el-2amar** |

- **Sun letters:** ت ث د ذ ر ز س ش ص ض ط ظ ل ن
- **Moon letters:** أ ب ج ح خ ع غ ف ق ك م ه و ي

> **Memory aid:** Say the first letter. Does it come from the **front of the mouth** (tongue near the teeth)? → sun letter. A full "l" before it then automatically feels wrong.

## Pronunciation (Syria)

- The "e" in **el-** is very short: el-beet sounds almost like **l-beet**.
- **Exception:** with sun letters the "e" is clearly audible, because the "l" drops: **esh-shams**, **et-tazkaret**.
