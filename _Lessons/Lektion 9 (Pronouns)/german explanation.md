# Syrisch-Arabisch: Lektion 9 – Die Familie (L 3aylet)

In dieser Lektion lernst du, wie man über die **Familie** spricht und wie **Besitzverhältnisse** (mein, dein, sein) durch Suffixe ausgedrückt werden.

## 1. Vokabeln: Die Familie
Im syrischen Dialekt gibt es spezifische Begriffe für Verwandte, wobei oft zwischen der väterlichen und mütterlichen Seite unterschieden wird.

| Begriff (Arabisch) | Deutsch              | Erklärung                                               |
| :----------------- | :------------------- | :------------------------------------------------------- |
| **Ahli**           | Meine Eltern         | Zusammengesetzt aus *imm* (Mutter) + *2ab* (Vater)      |
| **i5wati**         | Meine Geschwister    | Zusammengesetzt aus *2a5* (Bruder) + *i5t* (Schwester)  |
| **Jiddi / Sitti**  | Mein Opa / Meine Oma | Großeltern                                              |
| **3amm / 3ammet**  | Onkel / Tante        | Geschwister des Vaters                                  |
| **5al / 5alet**    | Onkel / Tante        | Geschwister der Mutter                                  |
| **ibn / Bint**     | Sohn / Tochter       | Wird auch für Cousin/Cousine genutzt (z.B. *ibn 3ammi*) |
| **Rfi2 / Rfi2at**  | Freund / Freunde     | Freundschaftliche Beziehungen                           |

---

## 2. Grammatik: Possessivsuffixe (Damir muttasil)
Anstatt separate Wörter wie "mein" oder "dein" zu verwenden, hängst du im Arabischen eine Endung direkt an das Nomen (z.B. *ism* = Name).

| Pronomen              | Suffix   | Beispiel | Deutsch    |
| :-------------------- | :------- | :------- | :--------- |
| **ana** (ich)         | **-i**   | ismi     | mein Name  |
| **inte** (du, m.)     | **-ak**  | ismak    | dein Name  |
| **inti** (du, w.)     | **-ek**  | ismek    | dein Name  |
| **hwwe** (er)         | **-o**   | ismo     | sein Name  |
| **hyye** (sie)        | **-a**   | isma     | ihr Name   |
| **ni7na** (wir)       | **-na**  | ismna    | unser Name |
| **intu** (ihr)        | **-kon** | ismkun   | euer Name  |
| **hinnin** (sie, pl.) | **-on**  | ismun    | ihr Name   |

---

## 3. Wichtige Fragen & Sätze
* **Mien hadol?**: Wer sind diese? 
* **Hada ana**: Das bin ich.
* **La2, hadol mo Jiddi w Sitti**: Nein, das sind nicht mein Opa und meine Oma.
* **Schu ismu 5alak?**: Wie heißt dein Onkel (mütterlicherseits)?
