# al- / Bestimmtheit – Referenz

> [!NOTE] Kernregel
> Arabisch hat **nur einen** Artikel: den bestimmten Artikel **el- / al-** (ال).
> Es gibt **keinen** unbestimmten Artikel.

| Form | Bedeutung |
| --- | --- |
| Nomen **+ el-** | bestimmt (*das/der/die*) |
| Nomen **ohne el-** | unbestimmt (*ein/eine*) |
| **Eigenname** (Ahmad, Damaskus) | schon von alleine bestimmt → kein el- |

- **ktaab** → *ein Buch* · **el-ktaab** → *das Buch*
- **beet** → *ein Haus* · **el-beet** → *das Haus*

## Sonnen- vs. Mondbuchstaben

Das **l** von el- verhält sich je nach erstem Buchstaben des Nomens unterschiedlich.

| | Regel | Beispiel |
| --- | --- | --- |
| **Sonnenbuchstabe** | „l" fällt weg, erster Buchstabe wird **verdoppelt** | el + shams → **esh-shams** |
| **Mondbuchstabe** | el- bleibt normal | el + 2amar → **el-2amar** |

- **Sonnenbuchstaben:** ت ث د ذ ر ز س ش ص ض ط ظ ل ن
- **Mondbuchstaben:** أ ب ج ح خ ع غ ف ق ك م ه و ي

> **Merkhilfe:** Sprich den ersten Buchstaben aus. Kommt er **vorne im Mund** (Zunge nahe den Zähnen)? → Sonnenbuchstabe. Es fühlt sich dann automatisch falsch an, ein volles „l" davor zu setzen.

## Aussprache (Syrien)

- Das „e" in **el-** ist sehr kurz: el-beet klingt fast wie **l-beet**.
- **Ausnahme:** Bei Sonnenbuchstaben ist das „e" klar hörbar, weil das „l" wegfällt: **esh-shams**, **et-tazkaret**.
