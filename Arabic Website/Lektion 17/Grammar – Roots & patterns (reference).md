# Roots & Patterns (Jazr & Wazn) – Reference

Arabic words are built from two parts:

> **Root** = the *meaning* (usually 3 consonants) · **Pattern (Wazn)** = the *function / word type*

You "pour" a root into a pattern and get a finished word.

## The placeholder f-3-l

Grammarians mark the three root letters with **f · 3 · l** (root letter 1 · 2 · 3).
Instead of `[R1]a[R2]a[R3]` you simply write **fa3al**.

## The most important patterns

| Pattern | Function | Example (root → word) |
| --- | --- | --- |
| **fa3al** | "he did X" (past tense) | k-t-b → **katab** (he wrote) |
| **maf3al** / **maf3alet** | a **place** where X happens | k-t-b → **maktab** (office); d-r-s → **madraset** (school) |
| **fa33aal** / **fa33aalet** | **person/machine** that does X | T-b-5 → **Tabbaa5** (cook); gh-s-l → **ghassaalet** (washing machine) |
| **faa3el** | the one **who** does X | k-t-b → **kaateb** (writer) |
| **maf3uul** | something X was **done to** | k-t-b → **maktoob** (letter); f-t-7 → **maftuu7** (open) |
| **mif3aal** | a **tool** you do X with | f-t-7 → **miftaa7** (key) |

## Example: one root, many patterns

**T-b-5** (idea of "cooking"): Taba5 (he cooked) · maTba5 (kitchen) · Tabbaa5 (cook)

> [!NOTE] Important
> **Not every pattern fits every root.** You can't combine them freely – some combinations aren't real words. You develop a feel for it over time.
