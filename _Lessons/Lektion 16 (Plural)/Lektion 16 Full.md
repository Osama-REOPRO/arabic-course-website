# Die drei Pluralarten

Im Arabischen gibt es **drei** Pluralarten, nicht nur eine wie im Deutschen.

## 1. Der gesunde maskuline Plural (Jam3 Mudhakkar Salim)

Diese Pluralform fügt **-iin** an das Wort an.

> mwazzaf + iin = **mwazzafiin** *(Angestellte)*
> suuriyy + iin = **suuriyyiin** *(Syrer)*

Diese Form benutzt man, wenn das Wort eine **männliche Person** beschreibt — egal ob es um einen Beruf, eine Herkunft oder einen Zustand geht.

<br>

> [!NOTE] Faustregel
> Beschreibt das Wort eine männliche Person (Beruf, Herkunft, Zustand)? → **-iin**. Gilt auch für gemischte Gruppen.
## 2. Der gesunde feminine Plural (Jam3 Mu2annath Salim)

Diese Pluralform fügt **-aat** an das Wort an, nachdem die Endung **-et/-at** (taa marbuta) entfernt wurde.

> mwazzafet → mwazzaf + aat = **mwazzafaat** *(Angestellte, weiblich)*
> sayyaarat → **sayyaaraat** *(Auto → Autos)*

Diese Form ist die vorhersehbarste von allen:
- Endet ein Wort auf **-et/-at** (taa marbuta)? Fast immer **-aat**: *mumarridat → mumarridaat*
- **Feminine Berufe/Herkunftsbezeichnungen** funktionieren genauso
- **Neue oder "importierte" Wörter** nehmen oft **-aat**, fast unabhängig vom Geschlecht — das ist auch ein guter Standard-Tipp, wenn man ein Wort nicht kennt

> [!NOTE] Faustregel
> Endet das Wort auf **-et/-at**? Fast immer **-aat**. Das ist auch ein sicherer Standardtipp bei neuen oder fremden Wörtern.

## 3. Der gebrochene Plural (Jam3 Taksiir)

Das ist die schwierigste, aber auch häufigste Pluralform — besonders bei **altem, einfachem Grundwortschatz**. Das Wort verändert sich **von innen**: die Vokale ändern sich, das Muster (Wazn) ist neu, auch wenn die Konsonanten gleich bleiben.

> walad → **wlaad**
> bint → **banaat**
> tifl → **atfaal**
> beet → **byuut**
> kalb → **klaab**

Hierher gehören:
- **Die meisten nicht-menschlichen/unbelebten Wörter**, die nicht auf -et/-at enden: *beet, ktaab*
- **Viele ganz grundlegende Personenwörter**, ärgerlicherweise auch: *walad → wlaad*, *bint → banaat* (banaat *sieht* aus wie ein gesunder femininer Plural, ist aber eigentlich eine gebrochene Form — der Wortstamm folgt nicht dem regelmäßigen Muster)
- **Tiere**: *kalb → klaab*

> [!NOTE] Wichtig
> Es gibt **keine einzelne feste Regel** für gebrochene Plurale — es hängt vom Wortmuster (Wazn) der Wurzel ab, und das Syrische hat diese Muster aus dem klassischen Arabisch geerbt. Muttersprachler "berechnen" das nicht, sie kennen es einfach — genau wie deutsche Sprecher "Kind → Kinder" nicht herleiten, sondern auswendig wissen.

---

# Entscheidungshilfe: Welcher Plural?

Wenn du ein neues Wort lernst und seinen Plural nicht kennst, kannst du so vorgehen:

1. **Endet das Wort auf -et/-at (taa marbuta)?** → sehr wahrscheinlich gesunder femininer Plural (**-aat**)
2. **Beschreibt das Wort einen männlichen Beruf/eine Herkunft/einen Zustand und endet auf einen Konsonanten?** → wahrscheinlich gesunder maskuliner Plural (**-iin**)
3. **Alles andere** (die meisten Grundwörter, Objekte, Tiere, viele Personenwörter) → wahrscheinlich gebrochen → **einfach mitlernen**

---

# Zusammenfassung: Welche Form bei welchem Wort?

| Pluralart | Endung | Typischer Gebrauch |
|-----------|--------|---------------------|
| Gesund maskulin | -iin | maskuline Berufe/Zustände, endet auf Konsonant |
| Gesund feminin | -aat | Wörter auf -et/-at, oft auch neue/importierte Wörter |
| Gebrochen | (kein festes Muster) | viele Grundwörter, Objekte, Tiere, manche Personenwörter |

# Grammatik: Plural + Possessivsuffix

Die Possessivsuffixe, die wir schon kennen, können auch an Pluralwörter angehängt werden — genau wie bei Singularwörtern.

> mumarridiin + i = **mumarridiini** *(meine Krankenpfleger)*
> i5wat + i = **i5wati** *(meine Geschwister)* — das kennen wir schon!
> sayyaaraat + na = **sayyaaratna** *(unsere Autos)*

	- Hadool kutubi

	- rif2aati dakaatrat

# Entscheidungshilfe - Welcher Plural benutze ich?

## Stufe 1: Gesunder femininer Plural (-aat)?

**Endet das Wort auf -et / -at (Taa Marbouta/weiblich)?**
**→ Ja** = -aat
> mwazzafet → mwazzafaat / sayyaara → sayyaaraat

**Kein -et / -at — ist das Wort ein Lehnwort?**
**→ Ja** = -aat
> telfoon → telfoonaat / kambyuutar → kambyuutaraat

**→ Nein** = Weiter zu Stufe 2.

---

## Stufe 2: Gesunde maskuline Plural (-iin ) oder gebrochener Plural?

Passt das Wort zu einem dieser Muster?
- fängt mit **mu-** an *(meistens ein Beruf: muhandes, mumarrid)*
- endet auf **-iyy** *(meistens eine Nationalität oder Zugehörigkeit: suuriyy, 3arabiyy)*
- folgt dem **fa33aal**-Muster *(jemand, der etwas beruflich tut: sawwa2, Tabba5, bayya3)*

**→ Ja** = -iin
**→ Nein** = gebrochener Plural — einfach mitlernen!

> [!NOTE] Wie Araber es lernen
> Arabische Muttersprachler hängen einfach **-iin** ans Wort und hören, ob es sich richtig anfühlt. Wenn ja — richtig! Wenn nicht — gebrochener Plural.
> Ehrlich gesagt kämpfen auch viele Araber selbst damit — das ist also kein Fehler, den nur Lernende machen.

# Kurz
**1. Taa Marbouta (-et/-at)?** → -aat ✓
**2. Lehnwort?** → -aat ✓
**3. mu- / -iyy / fa33aal?** → -iin ✓
**4. Nichts davon?** → gebrochener Plural — auswendig lernen!
