# Idafeh (Genitivverbindung) – Referenz

Die **Idafeh** verbindet zwei Nomen zu einer Einheit (wie das deutsche Kompositum *Schul-tasche*).

> **Hauptwort + Bestimmungswort** → Tabba5 + maT3am = **Tabba5 maT3am** (*ein Restaurantkoch*)

## Regeln

| Regel | |
| --- | --- |
| Das **erste** Wort bekommt **nie** el- | |
| Bestimmtheit steuert das **zweite** Wort | |
| zweites Wort **+ el-** → ganze Idafeh **bestimmt** | Tabba5 **el-**maT3am = *der Koch des Restaurants* |
| zweites Wort **ohne el-** → **unbestimmt** | Tabba5 maT3am = *ein Restaurantkoch* |
| Beim Sprechen **keine Pause** – alles fließt wie ein Wort | mudif-el-maT3am |

## Feminine Wörter (Endung -et/-at)

In der Idafeh wird das „t" **hörbar**. Vor el- fällt zusätzlich das „e" vor dem „t" weg.

| Form | Beispiel | Deutsch |
| --- | --- | --- |
| Wort allein | taalebet | eine Studentin |
| Idafeh **ohne** el- | taalebet jaam3at | eine Universitätsstudentin |
| Idafeh **mit** el- | **taalebt** ej-jaam3at | die Studentin der Universität |

> **Merke:** Vor el- hat das „e" keinen Platz mehr → es fällt weg: taleb**et** → tale**bt** el-…

## Beispiele

- baab beet → *eine Haustür* · baab **el-**beet → *die Haustür*
- meftaa7 sayyarat → *ein Autoschlüssel* · meftaa7 **es-**sayyarat → *der Autoschlüssel*
- ghurfet noom → *ein Schlafzimmer* · ghurft **en-**noom → *das Schlafzimmer*

*(Beim zweiten Wort gelten die normalen [[Grammatik – al- (Referenz)|Sonnen-/Mondbuchstaben-Regeln]].)*
