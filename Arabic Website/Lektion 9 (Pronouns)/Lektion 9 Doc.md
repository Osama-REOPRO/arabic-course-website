# Possessivsuffixe

Im Arabischen gibt es keine separaten "Possessivpronomen" (wie mein, dein, ihr usw.). Stattdessen werden **Suffixe** (Endungen) an das Wort angehängt. Diese Suffixe sind unten aufgeführt:

| Pronomen | Suffix | Beispiel | Deutsch |
| :--- | :--- | :--- | :--- |
| **ana** (ich) | **-i** | ismi | mein Name |
| **inte** (du, m.) | **-ak** | ismak | dein Name |
| **inti** (du, w.) | **-ek** | ismek | dein Name |
| **hwwe** (er) | **-o** | ismo | sein Name |
| **hyye** (sie) | **-a** | isma | ihr Name |
| **ni7na** (wir) | **-na** | ismna | unser Name |
| **intu** (ihr) | **-kon** | ismkon | euer Name |
| **hinnin** (sie, pl.) | **-on** | ismon | ihr Name |

<div style="page-break-after: always;"></div>

### Beispiel mit "3elet" (Familie) 
| Pronomen | Wort + Suffix | Deutsch           |
| :------- | :------------ | :---------------- |
| ana      | 3elti         | meine Familie     |
| inte     | 3eltak        | deine Familie (m) |
| inti     | 3eltek        | deine Familie (w) |
| hwwe     | 3elto         | seine Familie     |
| hyye     | 3elta         | ihre Familie      |
| ni7na    | 3eletna       | unsere Familie    |
| intu     | 3eletkon      | eure Familie      |
| hinnin   | 3elton        | ihre Familie (pl) |
> [!NOTE]
> Beachte, dass das „t“ (Taa Marbouta) am Ende von **3elet** hier zwingend ausgesprochen werden muss, wenn ein Suffix angehängt wird.
> 
> Außerdem wird das zweite „e“ im Wort „3elet“ bei bestimmten Pronomen entfernt (z. B. *3elti*). Dies wird in einer späteren Lektion genauer erklärt.
